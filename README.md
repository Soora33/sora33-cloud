# 微服务项目

## 项目概要

本项目使用 Spring Cloud 和 Spring Cloud Alibaba 构建微服务架构，涵盖了多个中间件和技术栈，旨在提供一个功能完备、可扩展的系统。目前正在初始开发中……

## 版本信息

- JDK: 17
- Spring Boot: 3.0.2
- Alibaba Cloud: 2022.0.0.0-RC2
- Spring Cloud: 2022.0.0
